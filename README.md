### บทคัดย่อ

โครงการ Synchro มีวัตถุประสงค์เพื่อช่วยบรรเทาและป้องกันปัญหาการขาดสมาธิที่เกิดจากการใช้สื่อสังคมออนไลน์มากเกินไป การหลุดโฟกัสจากการเรียนหรือกิจกรรมที่กำลังปฏิบัติรวมถึงความเบื่อหน่ายและความเครียดที่สัมพันธ์กับการเรียนรู้ ปัญหาเหล่านี้หากปล่อยไว้นานอาจส่งผลต่อ คุณภาพการเรียนรู้ ประสิทธิภาพการทำงาน และสุขภาพจิตในระยะยาว โดยเฉพาะในกลุ่มนักเรียนและนักศึกษาที่ต้องอาศัยสมาธิในการทำงานเชิงลึก (deep work) และการเรียนรู้ต่อเนื่อง

แนวคิดของโครงการคือการใช้ “เกมจังหวะ (Rhythm Game)” เป็นสื่อกลางในการฝึกสมาธิ โดยออกแบบให้ผู้เล่นต้องจดจ่อกับจังหวะดนตรี เวลา และตำแหน่งการกดปุ่มอย่างต่อเนื่อง การเล่นซ้ำจะช่วยเสริม การโฟกัสเฉพาะจุด (focused attention) และ การตอบสนองที่แม่นยำ (accurate response) ซึ่งเป็นทักษะที่สามารถถ่ายทอดกลับไปสู่การเรียนหรือการทำงานจริงได้

---

### ขั้นตอนการพัฒนา :
1. ทำการวางแผนซื้ออุปกรณ์ที่ต้องใช้ทั้งหมด
2. ออกแบบตัวเว็ปไซต์ทาง Figma และ ทำ CI
3. สร้างเว็ปไซต์ส่วน Frontend ที่แสดงข้อมูล Documentation ของโปรเจ็กต์และการแสดงผลเมื่อเชื่อม Arduino
4. ต่ออุปกรณ์ทั้งหมด จอ LCD เข้ากับ ESP32 ปุ่ม Push Button
5. เขียนโปรแกรม Upload ขึ้น ESP32 ให้สามารถแสดงผลเกมและการกดปุ่มตอบสนองกับจอ
6. สร้างเครื่องมืออัปเพลงใส่ SD Card adapters และ หน้าสำหรับสร้างแมพเกม Rhythm แบบ JSON สำหรับให้ AI ช่วยสร้างแมพจอให้
7. ปรับแก้โค๊ดและเชื่อม ESP32 เข้ากับ Websevers ทำให้สามารถนำค่าคะแนนจาก ESP32 มาแสดงและสรุปผลได้
8. ปรับแต่งให้สามารถอัปเพลงลง SD card ได้เพิ่มเพลงได้และปรับปรุงให้ตัวงานสมบรูณ์

---
### Input และ Output ของ Project นี้ Input :
#### Input : 
1. การกดปุ่มของผู้เล่น (4 ปุ่มสี)
2. เพลง (ไฟล์ MP3/WAV บน microSD)
3. Map/โน้ตของเพลง (เวลาที่ควรกด)
4. การตั้งค่าจากเว็บ (เลือกเพลง, ความยาก, ความเร็ว BPM)
#### Output : 
1. เสียงเพลงจากลำโพง
2. Combo, Accuracy (แสดงบนเว็บและ/หรือ LCD) สรุปสถิติการเล่นแต่ละเพลง

### สิ่งที่ได้ลองทำ :
1. ลองเชื่อมต่อจอ LCD TFT Color module เข้ากับ ESP32
2. การเชื่อม Push button ให้กดปุ่มตรงกับแสดงบนจอ LCD
3. การเชื่อมต่อ ESP32 เข้ากับ Webservers และส่งค่าจากการกดไป Webservers
4. การออกแบบจอสีและแสดงผลทางจอ LCD
5. การ Config ตัวเว็ปไซต์ให้สามารถเชื่อมกับ ESP32 
6. การอัปโหลดเพลงลงในเว็ปไซต์และสามารถใช้เพลงนั่นได้
7. การสร้าง Tool บนเว็ปไซต์สำหรับสร้างแมพปุ่มกด Rhythm และ การอัปเพลงลง SD card adapters
### อุปสรรคอะไรบ้างระหว่างทาง  :
1. ตอนแรกที่วางไว้จะใช้ Arduino UNO R4 ในการเชื่อม Webservers กับจอแต่ไม่สามารถทำได้
2. การต่อปุ่ม Push button เข้ากับ ESP32 ไม่ได้
3. การเชื่อม SD card Adapter แล้วหา SD card ไม่เจอ
4. การเชื่อมต่อ Webservers ที่มีปัญหา
5. ข้อจำกัดด้านอุปกรณ์ในการเชื่อมลำโพงและอัปเพลงลงเว็ปแล้วใส่เพลงลง SD card adapters
### แก้ไขอุปสรรคยังไง :
1. เปลี่ยน Arduino UNO R4 ไปใช้ ESP32 แทนเเพราะ Arduino R4 ค่อยข้างใหม่ไม่มีกรณีการศึกษา วิธีการสอนมาก ไม่รู้วิธีการเรียกใช้งาน ESP32 ในเ Arduino และ ESP32 มีตัวไกด์เชื่อมกับจอและกสนสอนเกี่ยวกับเชื่อม Webservers
2. เรียกรู้ว่า Breadboard มีการจ่ายไฟเพียงครึ่งเดียวทำให้ต้องเชื่อมไฟให้ครบทั้ง Breadboard
3. ตัว SD card Adapter มีปัญหาการต่อและจ่ายไฟไม่พอเปลี่ยนจาก 3 V เป็น 5 V
4. Config ตัว Javascript ในการเก็บ Cookkie และ การทำ Handshiking ในการเช็คสถานะการเชื่อม ESP32 กับเว็ปไซต์
5. การใช้ลำโพงกับ ESP32 ไม่เพียงพอและเสียงเบาจึงต้องมีการใช้ Amplifier GF 1002 ในการปรับเสียงและ SD Card เป็นตัวกลางในการเก็บเพลงที่อัพจากเว็ปไซต์
### ทำอะไรตามแผนไม่ได้บ้าง :
1. บอร์ด Arduino R4 เป็น ESP32
2. ใช้ ESP32 ในการเล่นเพลง
3. การทำเพลงในการกดหลายเพลง
4. การออกแบบหน้าจอเกมให้หลากหลาย
5. การทำงานแบบ Realtime อัพเดตค่าแบบ Realtime
